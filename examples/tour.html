html-collab · how to use it

Reviewable HTML.

A CLI and embedded runtime that turn any HTML report into a portable review artifact — comments, edits, and a brief you can paste back to the AI that wrote it.

Scroll ↓
This page is itself a review artifact. Press C to comment on selected text, E to suggest an edit. Your browser may ask where to save the review file; that's normal. It keeps your comments and edits inside this local .html file, not on a server.

In the browser

What reviewers do.

All in-file. No install, no account, no upload.

In browser

Comment

Select text → press C
Select text, press C, write a thread. Reply, mark resolved, reopen. Threads have stable IDs so the brief can deep-link back to them.
In browser

Suggest an edit

Select text → press E
Select text, press E. Replace it, insert after it, or delete it. Toggle Markup or Preview to see the change in context. The document only changes when someone accepts.
In browser

Autosave

Toolbar toggle
The first time you add feedback, your browser may ask for permission to save. Choose the review file and html-collab will keep writing your comments and edits into that same local .html as you work.

From the CLI

What the author does.

From a terminal. Scriptable, CI-friendly, agent-friendly.

CLI

Wrap

html-collab wrap
html-collab wrap report.html --out report.review.html. Embeds the original report verbatim and adds the review runtime. Reversible — unwrap returns the exact bytes.
CLI

Merge

html-collab merge
html-collab merge a.review.html b.review.html --out merged.review.html. Combines two parallel reviews into one file. Operations dedupe by ID — no central server picks a winner.
CLI · the loop closer

Brief

html-collab extract

html-collab extract merged.review.html --format markdown --out brief.md writes a Markdown summary of every thread and suggested edit, with anchor quotes, statuses, and deep links back to the file.

Paste the brief into the AI that generated your HTML and ask it to address the feedback. That's the loop this whole format was designed to close.

AI writes HTML → wrap → review → brief → AI revises
CLI

Unwrap

html-collab unwrap
html-collab unwrap report.review.html --apply-edits --out report.final.html. Strips the review chrome and writes the final document. Ambiguous edits fail instead of guessing.
